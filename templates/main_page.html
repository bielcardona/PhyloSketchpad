The Phylogeneticist's Sketckpad

Help

Interactive canvas:
  • Create node: Alt-Click on canvas
  • Select node: Click on the node
  • Create edge: Select source node and shift-click on target node. If you alt-shift-click on canvas a new node will be created as target node
  • Select edge: Click on the edge
  • Drag canvas: Click on canvas and drag
  • Zoom canvas: Use wheel button on mouse
Contextual menus: Right-clicking brings up a contextual menu:
  • On canvas: In this (main) contextual menu you can select which data will be displayed for networks and nodes, as well as adding new networks or trees given by their (e)Newick string or random trees
  • On nodes: You can (re)label the node, delete it, and get the information you asked for in the main contextual menu
  • On edges: You can delete the edge
  • On network labels: You get the information you asked for in the main contextual menu

About

The Phylogeneticist's Sketchpad is brought to you by the Bioinformatics group at the University of the Balearic Islands. Its main author is Gabriel Cardona.

This web-tool uses the python library phylonetwork.

This project is open-source and is hosted at github

If you find a bug, or want some new feature to be implemented, please fill in an issue on the github page of the project or send an e-mail to the author.
